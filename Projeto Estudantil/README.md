# Análise de Desempenho Estudantil e Fatores Socioeconômicos

Este projeto investiga os dados de desempenho de alunos em exames de matemática, leitura e escrita para entender como variáveis externas influenciam o sucesso acadêmico.

### Principais Descobertas (Insights)
* **Influência Familiar:** Alunos cujos pais possuem nível superior (Mestrado/Bacharelado) apresentam médias significativamente mais altas.
* **Habilidades Conectadas:** Existe uma correlação fortíssima (**0.95**) entre leitura e escrita, indicando que o domínio de uma potencializa a outra.
* **Preparação:** O curso preparatório para testes foi um diferencial decisivo, elevando a mediana das notas em todas as matérias.

### Tecnologias Utilizadas
* **Python** (Pandas, Seaborn e Matplotlib)
* **Google Colab** para o desenvolvimento estatístico.
